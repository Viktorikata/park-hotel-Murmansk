# Park-Hotel Murmansk — многостраничный сайт парк-отеля

Многостраничный адаптивный сайт для парк-отеля «Экспедиция» в Мурманской области.

Проект выполнен по макету из Figma и включает несколько страниц с разными секциями, адаптивной вёрсткой, интерактивными элементами и анимациями. Основной акцент сделан на точную реализацию дизайна, аккуратную структуру HTML/CSS, адаптивность и удобство дальнейшей интеграции с CMS.

## 📌 О проекте

Сайт представляет парк-отель и его услуги: номера, экскурсии, питание, банкетный зал, контакты и дополнительные информационные страницы.

В проекте реализованы:

- главная страница;
- страницы услуг и направлений;
- страницы с hero-блоками;
- секции с карточками, изображениями и декоративными элементами;
- интерактивные слайдеры/переключатели контента;
- адаптивная версия для мобильных устройств;
- повторяющиеся компоненты шапки и футера;
- кастомная страница 404;
- страница политики конфиденциальности.

## ✨ Что реализовано

### Адаптивная вёрстка

Сайт адаптирован под разные разрешения экранов, включая desktop и mobile-версии. Для мобильных устройств используются отдельные изображения и перестроение сеток.

### Компонентный подход в HTML/CSS

Код разделён на независимые секции и блоки. Используется БЭМ-подход, чтобы стили было проще поддерживать и переиспользовать.

Примеры блоков:

- `hero`
- `room-preview`
- `excursion-section`
- `special-offer`
- `site-header`
- `site-footer`
- `feedback`

### Интерактивные элементы

На JavaScript реализованы переключатели контента и изображений:

- смена слайдов;
- обновление текста, номера, заголовка и изображения;
- блокировка повторного клика во время анимации;
- плавные переходы через CSS-классы;
- работа с `data-*` атрибутами.

### Работа с изображениями

Изображения добавлены через тег `<img>`, а не через CSS `background-image`, чтобы в дальнейшем их было проще заменять через CMS.

Также добавлены:

- `alt` для значимых изображений;
- `aria-hidden="true"` для декоративных элементов;
- отдельные изображения для desktop и mobile.

### Доступность

В проекте используются базовые практики доступности:

- aria-label для интерактивных кнопок;
- пустой `alt` у декоративных изображений;
- семантичные элементы;
- корректные типы кнопок;
- понятная структура интерактивных блоков.
